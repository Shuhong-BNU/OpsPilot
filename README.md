## Positioning

OpsPilot is positioned as `基于 RAG 与 MCP 的智能运维助手` / `an intelligent operations assistant built on RAG and MCP`.

It focuses on:

- intent routing
- hybrid retrieval and reranking
- Plan-Execute-Replan diagnosis
- MCP tool orchestration
- persistent session and workflow state
- lightweight observability and testability
